Python Basics

print(), variables, input(), data types — un first step into Python

1 print() Function
S

Real-Life Example

Story
Tea kadai-la order solla

Nee oru tea kadai-ku poi, counter-la nikkura. Nee enna pannuva? "Oru tea kudunga" nu solluva. Nee sollama iruntha, kadaikkaaran unakku enna venum nu theriyaathu.

Computer-kku nee enna venum nu sollanumna, print() use pannanum. Nee sollaama iruntha, screen-la onnum varaathu. Tea kadai-la vaai thirakaama ninna maari.

C

Concept Explanation

Understanding

print() is a built-in function. Ithu un text-a screen-la display pannum. Nee kudukkura text-a quotation marks ("..." or '...') ulle podu, aprom print()-kku kudu.

Ithu verum display function. Calculate pannaathu, store pannaathu. Verum kaatthidum.

$

Python Syntax

How to write
Python 
print("Un message idhula podu")
  • print — function name. Lowercase la ezhuthanum
  • () — round brackets, function call sign
  • "..." or '...' — text-a ithula wrap pannu
>

Worked Example

Code + Output
Python 
print("Vanakkam, Python!")
print("Naan Python kaththukittu irukken.")
print("2 + 3 =", 2 + 3)
Output
Vanakkam, Python!
Naan Python kaththukittu irukken.
2 + 3 = 5
2 Variables
S

Real-Life Example

Story
Kitchen dabba-la label potta store panra maari

Un amma kitchen-la different dabbas vechirukkanga. "Arisi" label, "Paruppu" label, "Sakkarai" label. Dabba-thaan variable. Label-thaan variable name. Ulle irukka poruluthaan value.

Amma "paruppu kudu" nu sonna, nee "Paruppu" label irukka dabba-va edukkuva. Variable-la irukka value-yum maathalam.

C

Concept Explanation

Understanding

Variable = named storage. = sign use panni value assign pannu. Python automatic-a type decide pannidum.

  • Letter or underscore la start aaganum
  • Spaces, special characters koodaathu
  • Case-sensitive: Name and name are different
>

Worked Example

Code + Output
Python 
kadai_peyar = "Murugan Tea Stall"
tea_vilai = 15
bun_vilai = 10
total = tea_vilai + bun_vilai

print("Kadai:", kadai_peyar)
print("Total:", total)
Output
Kadai: Murugan Tea Stall
Total: 25
3 input() Function
S

Real-Life Example

Story
Bank-la form fill panra maari

Bank-la counter person kekkuvaan: "Peyar enna?" — nee solluvaa. "Phone number?" — nee solluvaa. Avan un answer-a form-la ezhuthuvan.

input() athaan ithu maari. Python kelvi kekum, nee type pannuva, Python variable-la store pannidum.

C

Concept Explanation

Understanding

input() always string return pannum. Number-a use pannanum na, int() or float() la convert pannu.

>

Worked Example

Code + Output
Python 
peyar = input("Unga peyar enna: ")
km = float(input("Evvalavu km: "))
total = km * 15
print(peyar, "- Total fare: Rs.", total)
Output
Unga peyar enna: Sanjay
Evvalavu km: 8.5
Sanjay - Total fare: Rs. 127.5
4 Data Types
S

Real-Life Example

Story
Ration card-la details maari

Ration card-la: Peyar (text), Vayasu (whole number), Weight (decimal), Card holder-ah (Yes/No). Different type data, different-a store pannanum.

C

Concept Explanation

Understanding
  • str — Text: "Vanakkam"
  • int — Whole numbers: 10, -5
  • float — Decimal numbers: 3.14
  • bool — True or False

type() function use panni type check pannalam.

>

Worked Example

Code + Output
Python 
peyar = "Murugan"
vayasu = 45
weight = 72.5
card = True

print(type(peyar))   # <class 'str'>
print(type(vayasu))  # <class 'int'>
print(type(weight))  # <class 'float'>
print(type(card))    # <class 'bool'>
!

Common Mistakes

Watch out
Quotes marakka (Forgetting quotes)
print(Vanakkam) # NameError print("Vanakkam") # Correct
Print nu capital la ezhuthurathu
Print("Hello") # NameError print("Hello") # Correct
input() return-a direct math-la use panrathu
age = input("Age: ") print(age + 5) # TypeError age = int(input("Age: ")) print(age + 5) # Correct
Variable-la space use panrathu
my name = "Sanjay" # SyntaxError my_name = "Sanjay" # Correct

Real-Life Problem

Q5: Auto fare calculator — From, To, km vaangi, base=Rs.25, per km=Rs.12, total calculate panni bill print pannu.

Expected Output
===== AUTO BILL =====
From: Tambaram
To: Guindy
Distance: 18.0 km
Base fare: Rs. 25
Per km: Rs. 216.0
Total: Rs. 241.0
=====================
Show Answer
Python 
s = input("From: ")
e = input("To: ")
km = float(input("Distance km: "))
base = 25
kmc = km * 12
print("===== AUTO BILL =====")
print("From:", s)
print("To:", e)
print("Distance:", km, "km")
print("Base fare: Rs.", base)
print("Per km: Rs.", kmc)
print("Total: Rs.", base + kmc)
print("=====================")
