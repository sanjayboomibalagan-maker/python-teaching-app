1 print() Function
S

Real-Life Example

Story
Tea kadai-la order solla

Nee oru tea kadai-ku poi, counter-la nikkura. Nee enna pannuva? "Oru tea kudunga" nu solluva. Nee sollama iruntha, kadaikkaaran unakku enna venum nu theriyaathu.

Computer-kku nee enna venum nu sollanumna, print() use pannanum. Nee sollaama iruntha, screen-la onnum varaathu. Tea kadai-la vaai thirakkama ninna maari.

🍵
C

Concept Explanation

Understanding

print() is a built-in function. Ithu un text-a screen-la display pannum. Nee kudukkura text-a quotation marks ("..." or '...') ulle podu, apparam print()-kku kudu.

Ithu verum display function. Calculate pannaathu, store pannaathu. Verum kaattum.

💡 Pro Tip: Terminal Check

Nee program run pannum pothu, output enga varumnu yosingira? Program output eppavum 'Terminal' or 'Console'-la thaan show aagum. print() koduthaa thaan terminal-la response kidaikkum!

🚀 Mastery Deep Dive: Advanced print()

Verum output mattum illa, print() function-la special powers-um irukku.

sep (Separator)

Multiple items-a eppadi separate pannanum nu solla. Default-a space irukkum.

print("A", "B", sep="-")
Output: A-B

end (End of line)

Print panni mudichona next line pogaama enna pannanum nu solla.

print("Hi", end="!!!")
Next print athe line-la varum.
