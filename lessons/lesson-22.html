2 SQL Commands in Python
C

Concept

Storage
  • Connection: Database file-ku connect pannum.
  • Cursor: Database-kulla query run panna use aagum.
  • execute(): SQL command-a (Create, Insert, Select) run panna.
  • commit(): Changes-a permanent-a save panna.
Python 
import sqlite3

conn = sqlite3.connect('mydata.db')
cursor = conn.cursor()

# Create Table
cursor.execute("CREATE TABLE IF NOT EXISTS users (name TEXT, age INTEGER)")

# Insert Data
cursor.execute("INSERT INTO users VALUES ('Sanjay', 22)")
conn.commit()

Quick Quiz – Database Basics

Data-va lock pannitiga-nu check pannalam!

1. Database-la pannuna changes-a (Insert/Update) save panna enna call pannanum?

save()
commit()
Explanation: **commit()** pannina thaan changes physical file-la write aagum.

2. Database queries-a execute panna thevaiyana object enna?

Cursor
Pointer
Explanation: **Cursor** thaan database commands-a carry panni execute pannanum.

3. Database-la irundhu data-va read panna enna SQL keyword use pannanum?

GET
SELECT
Explanation: **SELECT** thaan data-va view panna use aagura standard SQL command.

4. SQLite file-a create/open panna enna function use pannanum?

open()
connect()
Explanation: sqlite3.connect() thaan database-oda logic-a start pannanum.

5. SQLite-ku extra installation (pip) thevaiya?

Yes, venum
No, standard library-la irukku
Explanation: SQLite3 Python library-oda built-in-aave varum.
