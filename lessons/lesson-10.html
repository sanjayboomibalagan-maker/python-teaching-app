Lesson 10

Functions & Return

Code Reusability — orey logic-a thirumba thirumba ezhuthama oru module-a mathu!

1 Function Basics
S

Real-Life Example

Story
Coffee Vending Machine

Oru machine irukku. Nee "Coffee" button-a amukuna, athu milk, sugar, and powder-a mix panni unakku coffee-a kudukkum. Machine kulla enna nadakuthunu unnakku theriya venam, aana "Input" (Button push) kudutha "Output" (Coffee) varum. Ithu thaan Function — oru specific task-a panna ready-a ezhuthi vachirukura code.

C

Concept

Understanding

Python-la function-a def keyword vachi start pannuvom.

  • def: Define function.
  • Parameters: Function-kku ulla anuppura inputs.
  • return: Function vela mudicha apparam kudukkura pathil (Output).
  • Call: Function-a use panna call pannanum (e.g., `greet()`).
Python 
# Define pannuthal
def greeting(name):
    print(f"Vanakkam {name}! Eppadi irukiga?")

# Call pannuthal
greeting("Sanjay")
greeting("Raja")
2 Parameters vs Arguments
C

Inputs in Detail

Logic
  • Parameter: Function define panra appo ezhuthura variables (e.g., `def add(a, b)`).
  • Argument: Actual call-la kudukura values (e.g., `add(5, 10)`).
  • Default Arguments: Argument kudukkalana automati-a eduthuka vendiya default value.
Python 
def welcome(name, city="Chennai"):
    print(f"Welcome {name} from {city}")

welcome("Arun")  # Arun from Chennai (Default)
welcome("Vijay", "Madurai")  # Vijay from Madurai
3 The 'return' Keyword
S

Value Returning

Result
Calculator Result

Nee calculator-la `2 + 2` podura. Athu screen-la result-a mathira unakku thiruthipum "4" nu oru value kudukum. Itha thaan return-um pannum — block-la calculate panna value-a veliya oru variable-kaku anuppum.

Python 
def calculate_gst(price):
    tax = price * 0.18
    return tax

# Value-a variable-la store pannalam
my_tax = calculate_gst(1000)
print("Tax amount:", my_tax)
