1 Async vs Sync
S

Real-Life Example

Story
Coffee Shop Multitasking (The Power of Await)

Nee oru coffee shop waiteru. Sync programming na, nee coffee machine-a on panni coffee vara varai andha machine munnadiye nipppa (**Waiting**). Async na, coffee machine on pannitu, andha gap-la innoru customertta order vaanguva (**Working**). Machine "Ding" sound kuduthona (Event), nee poi coffee eduppa. Idhu thaan Async logic. Network requests mudiya wait pannaama program execution-a continue pannanum.

2 Await and Sleep
C

Concept

Syntax
  • async def: Definition of a coroutine (non-blocking function).
  • await: Oru result varra varai execution-a pause panna (without blocking others).
  • run(): Main async entry point-a trigger panna.
Python 
import asyncio

async def make_coffee():
    print("Starting coffee...")
    await asyncio.sleep(2) # Non-blocking wait
    print("Coffee ready!")

asyncio.run(make_coffee())

Quick Quiz – Non-blocking Python

Speed logic catch pannitiga-nu check pannalam!

1. Oru async function kulla vera oru async task mudiya wait panna enna keyword pannanum?

wait
await
Explanation: **await** keyword thaan coroutine context-a release pannanum so others can run.

2. `async def` vachi define pannura function-a enna-nu soluuvom?

Coroutine
Generator
Explanation: Async functions-a **Coroutines** nu solluvom.

3. Async-la program block aagaama 'wait' panna endha function use aagum?

time.delay()
asyncio.sleep()
Explanation: **asyncio.sleep()** thaan non-blocking sleep-a proivde pannanum. `time.sleep` whole program-a block pannanum.

4. Top-level async task-a start panna enna call pannanum?

asyncio.start()
asyncio.run()
Explanation: **asyncio.run()** thaan event loop-a start panni coroutine run pannanum.

5. Async programming high network-usage apps-ku (like Chat / Crawlers) romba useful-ah?

Yes, romba useful
No, support aagaathu
Explanation: I/O bound tasks-ku (Internet/File) **Async** is extremely fast because CPU free-a irukkum while waiting for data.
