Multithreading & Multiprocessing

Parallel Programming — Orey nerathula neraya vela seiya vai! Time-a save pannu.

1 Multithreading vs Multiprocessing
S

Real-Life Example

Story
Hotel Kitchen (Samayal Room)

Oru hotel-la Multithreading-nu sonna, oru chef-ae orey nerathula tea-um pottu, side-la vadai-um fry panra maari. Ithu "Lightweight" tasks-ku (like waiting for internet download) semma speed-a irukkum.

Multiprocessing-nu sonna, 4 periya chefs-a oru periya function-ku (Kalyana saapadu) hire panra maari. Ovvoru chef-kum oru "Separate Stove" (CPU Core) irukkum. Ithu romba "Heavy" tasks-ku (like video editing or heavy math) use aagum.

2 How it works?
C

Concept

Concurrency
  • Thread: Oru process-kulla irukura chinna stream of execution. Multiple threads same memory space-a share pannum.
  • Process: Engine maari. Independent memory block. CPU-la romba cores iruntha multiprocessing semma speed-a irukkum.
Python (Threading) 
import threading
import time

def print_numbers():
    for i in range(5):
        print(i)
        time.sleep(1)

# Creating thread
t1 = threading.Thread(target=print_numbers)
t1.start()
t1.join() # Wait for it to finish

Quick Quiz – Parallel Power

Speed-a kathukitiga-nu check pannalam!

1. Net-la irundhu file download aakura appo UI lag aagame irukka enna use pannanum?

Simple Loop
Multithreading
Explanation: Waiting tasks (I/O Bound) tasks-ku **Multithreading** thaan best choice.

2. Multiprocessing use panna hardware-la enna extra-va theva padum?

More RAM
Multiple CPU Cores
Explanation: Multiprocessing parallel-a run aaga computer-la multiple cores (engines) irukkanum.

3. Oru thread mudiyura varaikkum main program-a wait panna vaikka enna method call pannanum?

t1.stop()
t1.join()
t1.wait()
Explanation: **join()** method thaan control-a hold panni thread finish aaga wait pannum.

4. Python-la 'GIL' na enna?

General Interpreter Loop
Global Interpreter Lock
Explanation: **GIL** thaan Python-la orey nerathula multiple threads CPU cores use panratha restrict pannum (Cpython logic).

5. Real parallel processing (true CPU parallelism) thevanna enna use pannanum?

Threading
Multiprocessing
Explanation: Multiprocessing thaan different cores use panni unmaiyana parallel speed kudukkum.
