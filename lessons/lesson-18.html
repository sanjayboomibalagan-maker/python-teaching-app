2 Map & Filter
C

Concept

Processing
  • Map: Oru list-la irukura ellathukum oru operation panna (e.g., Ellaa items-um double aakka).
  • Filter: List-la irukura certain items-a mattum filter panni edukka.
Python 
nums = [1, 2, 3, 4, 5]

# Map: Square all
sqs = list(map(lambda x: x*x, nums))

# Filter: Even numbers only
evens = list(filter(lambda x: x % 2 == 0, nums))

print(sqs)
print(evens)

Quick Quiz – Lambda & Tools

Functional magic catch pannitiga-nu check pannalam!

1. Lambda function and innoru peyar enna?

Anonymous function
Explanation: Name illama use panrathala idhu Anonymous function.

2. List-la irukura ovvoru element mela oru function-a apply panna enna use pannanum?

map()
filter()
apply()
Explanation: map() function input list-a output list-a map pannanum applying the operation.

3. Filter function-kulla kudukura lambda eppovum enna return pannanum?

Number
Boolean (True/False)
Explanation: Condition True-va iruntha mattum thaan element results-la varum. So Boolean return pannanum.

4. Lambda functions-la max-a ethana line expression ezhutha mudiyum?

One single line
Many lines
Explanation: Lambda strictly **one-line** functions thaan. Complex logic-ku def use pannanum.

5. map() or filter() oda result-a readable-a pakka enna type-a convert pannanum?

string
list()
Explanation: Python 3-la map/filter object-a thaan tharum. Athanaala list() panna thaan values-a pakka mudiyum.
