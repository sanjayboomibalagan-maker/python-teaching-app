# Linear Search
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# Bubble Sort logic
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

2. Bubble sort-oda worst-case complexity enna?

Explanation: Double loop irukkradhaala complexity **O(n^2)** aayidum.

3. Binary search panna list mandatory-a epdi irukkanum?
