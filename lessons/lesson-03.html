Lesson 03

Strings & String Methods

Text handling, slicing, f-strings, upper(), split(), join() — text-oda full control

1 String Basics
S

Real-Life Example

Story
Name board-la ezhuthurathu maari

Kadai-la name board irukkum: "Murugan Mess". Antha text-la nee letters mattum edukkalaam, reverse pannalaam, capital-la maathalam. Python-la string-um ithe maari — text-a store panni, manipulate pannalam.

🔤
C

Concept

Understanding

String = text data. Quotes ulle irukkura ellaamey string thaan.

  • Single quotes: 'Hello'
  • Double quotes: "Hello"
  • Triple quotes (multiline): '''Hello World'''

💡 Pro Tip: Immutability

Strings are immutable. Ithu rumba simple concept thaan — oru thadava string create pannitta, athula irukkura elements-a change panna mudiyathu. Change pannanum na puthu string thaan create pannanum!

>

Worked Example

Code
name[0:3]
String / Var name
Start index (Include) 0
End index (Exclude) 3
Python 
name = "Chennai"
print(name[0])      # C (first character)
print(name[-1])     # i (last character)
print(name[0:3])   # Che (slicing)

🚀 Mastery Deep Dive: Advanced Slicing

Slicing-la 3 params irukku: [start : stop : step].

The 'Step' parameter

Oru letter vittu oru letter edukka step=2 use pannalaam.

"Python"[::2]
Output: Pto

String Reversal

Reversing logic in one line!

"Chennai"[::-1]
Output: iannehC
2 String Methods
C

Important Methods

Understanding
Upper.upper()
Lower.lower()
Strip.strip()
Split.split()
Replace.replace()
Python 
msg = "  Vanakkam Chennai  "
print(msg.strip())            # "Vanakkam Chennai"
print(msg.strip().upper())    # "VANAKKAM CHENNAI"
print(msg.strip().replace("Chennai", "Madurai"))
# "Vanakkam Madurai"

fruits = "apple,mango,banana"
print(fruits.split(","))  # ['apple', 'mango', 'banana']

words = ["Naan", "Python", "padikirein"]
print(" ".join(words))  # "Naan Python padikirein"
3 f-Strings (Formatted Strings)
S

Real-Life Example

Story
Wedding invitation card-la name fill panra maari

Invitation card template irukkum: "We invite _____ to the wedding." Blank-la name fill pannuva. f-string-um athe thaan — template-la variable values fill pannum.

💡 Pro Tip: f-string Magic

Quotes-kku munnadi oru f pottom na, string ulla curly brackets {} use panni endha variable or expression-a venum naalum direct-a insert pannalaam. Ithu rumba neat and clean approach!

>

Worked Example

Code
Python 
name = "Sanjay"
age = 21
city = "Chennai"

# f-string: f"..." ulle {variable} podu
print(f"peyar: {name}, vayasu: {age}, ooru: {city}")
print(f"Next year age: {age + 1}")
print(f"Name uppercase: {name.upper()}")
Output
peyar: Sanjay, vayasu: 21, ooru: Chennai
Next year age: 22
Name uppercase: SANJAY

🚀 Mastery Deep Dive: f-string formatting

Precision Control

Decimal points control panna: {val:.2f} use pannuvaanga.

pi = 3.14159; f"{pi:.2f}"
Output: 3.14

Expressions inside

Curly brackets ulla calculation/methods-um use pannalaam.

f"{name.upper()} counts {len(name)}"
!

Common Mistakes

Watch out
String index out of range
"Hello"[10] # IndexError "Hello"[4] # o (last valid index)
String immutable-a marakka
s = "Hello" s[0] = "J" # TypeError s = "J" + s[1:] # "Jello" (new string create pannu)
f-string la f marakka
print("Name: {name}") # literal {name} print aagum print(f"Name: {name}") # actual value print aagum
