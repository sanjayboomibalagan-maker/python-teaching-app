Real-Life Example

Story
Police Investigation (Fingerprints)

Oru crime scene-la neraya fingerprints irukkum. Police kitta oru "Pattern" irukkum (Suspect-oda fingerprint). Neraya fingerprints-la endha fingerprint suspect-oda pattern-ku match aagudhu-nu check pannuvanga. Ithu thaan Regex. 2 Common Patterns
C

Concept

Syntax
  • \d: Digit (0-9) kandu pudikka.
  • \w: Alphanumeric (Letters + Numbers).
  • +: One or more (Neraya digits iruntha e.g., \d+).
  • findall(): Oru text-la match aagura ellathaiyum list-a thandhudum.
Python 
import re

text = "My order ID is 4567 and my bill is 120."
nums = re.findall("\d+", text)

print(nums) # Output: ['4567', '120']

4. Oru text pattern-a Vera text-a replace panna endha function venum?

Explanation: Regex module-la sub() (Substitute) thaan replacement-ku use pannuvom.
