Exception Handling

Error Management — Code crash aagama kapathi, safety-a handle pannu!

1 try ... except
S

Real-Life Example

Story
Safety Fuse (Electric Circuit)

Namma veetla current overload aana circuit breakdown aagi fuse poyidum. Ithu vee-la irukura fridge, tv-a safe-a vachukka thaan. Ithu maari program-la ethavathu thappu nadantha (e.g., divided by zero), program crash aagi stop aagidum. try...except use panna, error vanthaalum program-a crash aagama "Safety plan"-kku kondu pogalam.

2 Common Exceptions
C

Concept

Logic
  • try: Error irukkumanu check panna vendiya code block.
  • except: Error vantha enna pannanum nu solra block.
  • finally: Error vanthaalum varalaatiyum kandippa nadakanum nu solra code block.
Python 
try:
    num = int(input("Enter a number: "))
    result = 10 / num
    print(result)
except ZeroDivisionError:
    print("Zero-vaala divide panna mudiyathu!")
except ValueError:
    print("Correct-aana number-a type pannu nanba!")
finally:
    print("Operation Finished.")

🚀 Mastery Deep Dive: Robust Error Logic

else Block

Try-la endha error-um varalana mattum oru block run aaganumna else use pannalam. Ithu except block-ku apparam ezhuthanam.

raise Keyword

Nee software ezhuthum pothu "Ithu thappu!" nu neeyae oru error-a uruvaakka raise use aagum.

if age < 0: raise ValueError("Age negative-a irukka mudiyaathu!")

Custom Exceptions

Python-la irukura errors mattum illama, unakunu pudhu error (e.g. LowBalanceError) create pannalam. Ithu OOPs class inheritance logic-la vela seiyum.

Quick Quiz – Error Handling

Program-a safe-a protect panna kathukitiga-nu check pannalam!

1. Error vantha execution handle panna enna keyword venum?

error
except
catch
Explanation: Python-la errors handle panna except thaan keyword. Java/JS-la 'catch' use pannuvanga.

2. Error vanthaalum varalaatiyum kandippa run aaga vendiya block enna?

always
finally
last
Explanation: File close panrathu maari 'Must run' tasks-a finally block-la ezhuthuvom.

3. "123"-ku bathula "Hello"-va int-a matha paatha enna error varum?

TypeError
ValueError
Explanation: Value incompatible-a (Type sari aana format thappu) iruntha ValueError varum.

4. Endha control handling-um illana error vantha program enna aagum?

Slow-aagum
Stop aagidum (Crash)
Ignore pannidum
Explanation: Python-la uncaught exceptions direct-a program-a terminate (crash) pannidum.

5. 10 / 0 nu ezhuthuna enna error varum?

MathError
ZeroDivisionError
Explanation: Mathematics rules padi zero-vaala divison possible illa, athu ZeroDivisionError.
