1 Coding Best Practices (PEP8)
C

Clean Code

Guidelines

Code ezhuthuradhu mattum mukkiyamm illa, adhu innooru person-ku puriyura maari ezhuthanum. Python-la standard coding style-a **PEP8**-nu solluvom.

  • Indentation: Always use 4 spaces.
  • Naming: Variables-ku `snake_case`, Classes-ku `PascalCase` use pannanum.
  • Documentation: Docstrings (`"""Docstring"""`) vachi functions-a explain pannanum.
  • Refactoring: Orer logic-a thirumba thirumba ezhuthaama, functions-a break pannanum (**DRY - Don't Repeat Yourself**).
