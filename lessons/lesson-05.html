2 while Loop
S

Condition Basis

Story
Mobile Charging

Mobile charge la potruka. WHILE Battery < 100%, charge yeritey irukanum. Eppo 100% aagutho (condition false), appo charging stop aagum. Inga count mukkiyam illa, condition thaan mukkiyam.

🔋

💡 Pro Tip: Increment check

While loop use pannum pothu variable-a update panna maranthuraatheenga (e.g. i += 1). Appadi update pannaalathaan condition false aagi loop stop aagum. Illana 'Infinite Loop' vandhudum!

Python 
battery = 95

while battery < 101:
    print(f"Charging... {battery}%")
    battery += 1  # Increment

print("Full Charge! Battery safe.")
3 Loop Control — break, continue, pass
C

Controlling the Loop

Keyword Logic
  • break: Loop-a appadiyae stop panni veliya vanthuru.
  • continue: Intha oru step-a mattum skip panni next iteration-kku po.
  • pass: Summa oru placeholder (mela ezhuthumbothu loop empty-a irukka koodathunu use panrathu).
Python 
# 1 to 10 print pannu, aana 5 vantha stop pannu
for i in range(1, 11):
    if i == 5:
        break
    print(i)

# Odd numbers mattum skip panna (continue use case)
for i in range(1, 11):
    if i % 2 != 0:
        continue
    print("Even:", i)

🚀 Mastery Deep Dive: Loop Else Logic

Python-la for loop kooda horizontal-a oru else block podalaam. Ithu eppo execute aagum na, loop full-a suththi mudichona.

The break Exception

Loop ulla break execute aagi veliya vantha, else block run aagaathu. Loop normal-a mudinja thaan run aagum. Ithu searching algorithms-la romba useful!

Common Mistakes

Watch out
Infinite Loop (While Loop-la marakka)

Condition eppovume true-va iruntha program hang aagum.

i = 1 while i <= 5: print(i) # increment (i += 1) ezhuthala while i <= 5: print(i) i += 1
range() end value

range(1, 5) na 5 varaathu, 4 vara thaan varum.

range(5) # 0, 1, 2, 3, 4 range(6) # 0, 1, 2, 3, 4, 5
