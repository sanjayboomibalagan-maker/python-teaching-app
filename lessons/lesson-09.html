Lesson 09

Sets

Unique Collections — duplicates-kku idam ellaatha kootu!

1 Set Basics
S

Real-Life Example

Story
College ID Card System

Oru college-la 1000 students irukaanga. Ovvorutharukkum oru "Roll Number" irukkum. Orey roll number rendu perukku vara mudiyaathu. Intha roll numbers-a vera enna order-la vena store pannalaam, aana duplicates matum varave koodathu. Ithu thaan Set.

C

Concept

Understanding

Set curly braces { } ulla items-a comma separate panni ezhuthanum (aana key:value irukaathu).

  • Unique: Duplicates allow pannaathu.
  • Unordered: Items index order-la thangathu.
  • Unindexed: S[0] nu edukka mudiyaathu.
  • Mutable: Items-a add panni remove pannalam.
Python 
my_set = {1, 2, 3, 3, 3}
print(my_set)  # {1, 2, 3} - Duplicates auto-a poirum

# Add items
my_set.add(4)
my_set.remove(1)
2 Set Operations (Math in Python)
C

The Power of Sets

Math Operations

Sets use panni rendu collections-a compare panna mudiyum.

  • Union (|): Rendu set-aiyum serthu single set aakkum.
  • Intersection (&): Rendu set-layum common-a irukura items mattum edukkum.
  • Difference (-): Oru set-la mattum irukura items-a edukkum.
Python 
cricket = {"Raja", "Sanjay", "Kumar"}
football = {"Kumar", "Arun", "Vikram"}

# Common students (Intersection)
print(cricket & football)  # {"Kumar"}

# All unique students (Union)
print(cricket | football)

🚀 Mastery Deep Dive: Set Theory & Optimization

O(1) Lookup

Lists-la oru item irukkanu check panna full list-um search pannanum. Aana Set-la in operator instant-a pathil sollum. Ithu "Hashing" logic-la vela seiyum.

FrozenSets

Set mutable, so item change pannalam. Aana unakku mudiyaathu-nu fix panna frozenset() use pannalam. Itha dictionary keys-a kooda use pannalam!

Symmetric Difference (^)

Intersection-ku opposite. Rendu set-layum common-a illaatha items-a mattum edukkum.

{1, 2} ^ {2, 3} # Result: {1, 3}
