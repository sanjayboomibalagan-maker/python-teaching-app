Lesson 17

List Comprehension

Python Magic — 5 lines of code-a single line-la mudiyu! Short and Sweet.

1 The Logic
S

Real-Life Example

Story
Filtering Stones from Rice

Nee arisila iruntha kallu (stones) edukkanum. Normal-a oru oru kallu-a eduthu veliya vaipa. Ithu normal for-loop maari. Aana un kitta oru magic filter shop irukku, atha rice mela vachitona, automatic-a nalla arisi mattum separate aagi vandhudum. Ithu thaan List Comprehension — short signature syntax use panni list-a filter/modify pannalam.

💡 Quick Concept Check

List comprehension start/end panna enna bracket use pannanum?

new_list = x for x in nums
2 Syntax Breakdown
C

Concept

Speed

Nalla gavanichuko: [expression for item in iterable if condition]

  • expression: Result-la enna varunaum (e.g., x*x).
  • if condition: Optional filtering (e.g., x > 5).
Python 
nums = [1, 2, 3, 4, 5]

# Old way (4 lines)
sq = []
for x in nums:
    sq.append(x*x)

# Python way (1 line!)
sq_new = [x*x for x in nums]
print(sq_new)

🚀 Mastery Deep Dive: Advanced Comprehensions

Nested Comprehension

Oru list-kulla innoru list build panna (Matrix conversion) nested syntax use pannalam.

[[x for x in range(3)] for y in range(3)]

if-else logic

Filtering mattum illaama, value-a decide panna if-else combined-a use pannalam. Ippo syntax order marum!

[x if x>0 else 0 for x in nums]

Readability vs Performance

Performance wise comprehension best, aana logic romba complex-a iruntha normal loop ezhuthurathu thaan "readable" architecture-ku nallathu.
