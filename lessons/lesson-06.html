Lesson 06

Lists & List Methods

Collections of data — multiple items-a store panna lists thaan easy way

1 List Basics & Creation
S

Real-Life Example

Story
Grocery List (Maligai Porul List)

Kadaikku poga munnaadi paper-la oru list ezhuthuva: "Pal, Muttai, Bread, Arisi". Intha list-la items-a nee order-la vachirupa. 1st item ethu? 3rd item ethu nu unnakku theriyum. Python-la ithe thaan List.

🛒
C

Concept

Understanding

List square brackets [ ] ulla comma separate panni ezhuthanum. Ithu oru collection maari!

fruits = ["Apple", "Mango"]
List Name fruits
Brackets [ ]
Items "Apple", "Mango"

💡 Pro Tip: Heterogeneous Data

Python List-la nee mixed data types-um store pannalam. Ore list ulla String, Integer, and Boolean-um irukkalam.
L = ["Sanjay", 25, True] — ithu valid!

Python 
fruits = ["Apple", "Mango", "Banana"]
print(fruits[0])     # Apple
print(fruits[0:2])   # Slice (Apple, Mango)
print(len(fruits))    # 3 items

🚀 Mastery Deep Dive: Copying vs Referencing

List-a innoru variable-ku assign panna, Python puthu list create pannathu. Orey box-kku vera peyar thaan vekkum.

The Reference Problem

L1 = [1, 2]; L2 = L1

Ippo L2-a mathuna L1-um maaridum! Rendum orey memory-a point pannuthu.

Safe Copying

Puthu box venum na .copy() method use pannanum.

L2 = L1.copy()
2 Adding & Removing Items
C

List Methods

Essential Methods
Add End.append()
Add Pos.insert()
Remove Item.remove()
Remove Last.pop()
Sort A-Z.sort()
Python 
super_heroes = ["Ironman", "Spiderman"]
super_heroes.append("Batman")    # ["Ironman", "Spiderman", "Batman"]
super_heroes.insert(0, "Thor")    # Thor first varun
super_heroes.remove("Spiderman")
super_heroes.sort()              # A-Z arrange
3 Iterating over Lists
S

Loops with Lists

Logic
Checking Attendance

Attendance register-la names irukkum. Techer ovvonu name-a kooptu check pannu vaanga. Python-la list-ulla irukkura items-a for loop use panni access pannalam.

Python 
students = ["Raja", "Kumar", "Sanjay"]

for s in students:
    print("Welcome,", s)

🚀 Mastery Deep Dive: List Comprehension

Oru loop-um, list filter-um orey line-la ezhutha ithu thaan ultimate technique.

Compact Code

numbers = [i*i for i in range(5)]
Output: [0, 1, 4, 9, 16]

With Condition

evens = [x for x in range(10) if x%2==0]

Intha maari ezhuthuna unga code "Pythonic"-a irukkunu ellarum solve paanga!

!

Common Mistakes

Watch out
Index Out of Range
L = [1, 2] print(L[5]) # Error! Only 0, 1 possible print(L[1]) # Correct
append() vs extend()

append full list-a ore item-a add pannum. extend inside items-a add pannum.

L = [1, 2] L.append([3, 4]) # [1, 2, [3, 4]] - Nested! L.extend([3, 4]) # [1, 2, 3, 4] - Joined!

Real-Life Problem

Q5: List-la irukkura numbers-la "Largest Number" kandupidi loops use panni (don't use max()).

Show Answer
Python 
L = [45, 67, 8, 92, 34]
big = L[0]
for n in L:
    if n > big:
        big = n
print("Largest:", big)
