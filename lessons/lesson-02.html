Power (2^3)**

💡 Pro Tip: / vs //

Normal division / eppavum decimal (point) value-va thaan tharum. Ungalukku point illama integer mattum venum na // (Floor Division) use pannunga!

💡 Quick Concept Check

Multiply 10 with 5, and print the result.

print(10 5)
>

Worked Example

Code
Python 
a, b = 17, 5
print("Add:", a+b)    # 22
print("Div:", a/b)    # 3.4
print("Floor:", a//b) # 3
print("Mod:", a%b)    # 2
print("Power:", a**2)  # 289
