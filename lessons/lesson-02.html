Lesson 02

Operators & Expressions

Arithmetic, Comparison, Logical, Assignment — Python-la math and decisions

1 Arithmetic Operators
S

Real-Life Example

Story
Kadai-la bill calculate panra maari

Tea=15, Bun=10, Bajji=20. Total? 15+10+20=45. Friend-ku share? 45/2=22.50. Ithu thaan arithmetic.

C

Concept

Understanding
  • + Addition | - Subtraction | * Multiplication
  • / Division (float) | // Floor division (int) | % Modulus (remainder)
  • ** Power — 2**3 = 8
>

Worked Example

Code
Python 
a, b = 17, 5
print("Add:", a+b)    # 22
print("Div:", a/b)    # 3.4
print("Floor:", a//b) # 3
print("Mod:", a%b)    # 2
print("Power:", a**2)  # 289
2 Comparison Operators
S

Real-Life Example

Story
Exam pass/fail check

35 marks-kku mela pass. Nee 42 vaanguna: 42>=35 = True = pass!

C

Concept

Understanding
  • == Equal | != Not equal
  • > Greater | < Less | >= Greater or equal | <= Less or equal
Python 
marks = 42
print(marks >= 35)  # True
print(marks == 100) # False

Real-Life Example

Story
College admission eligibility

Admission-ku: 12th marks>=60 AND entrance pass. Rendu-um True na thaan admission. Scholarship-ku: Sports OR merit — onnaavathu True na podhum.

Worked Example

print(not False)  # True
