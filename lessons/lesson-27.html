Data Structures (Stacks & Queues)

Efficiency — Million bytes of data-va memory-la epdi correct-a arrange pannanum?

1 Stacks vs Queues
S

Real-Life Example

Story
Cafeteria Plates vs Bank Line

1. Stack: Oru plate mela innooru plate adukki vacha, namma first edukkumbothu last-a vacha plate-a thaan eduupom (**LIFO - Last In First Out**).

2. Queue: Bank line-la first yaaru nilluranga thaan first service kedaikum (**FIFO - First In First Out**). Python-la lists-a vachi idhe structures-a create panna kathukanum.

💡 Quick Concept Check

"First In First Out" behavior-ku logic peyar enna?

Pattern:
2 Implementation
C

Concept

Logic
  • Push: Oru element-a add panna (List-la `.append()`).
  • Pop: Oru element-a remove panna (List-la `.pop()`).
  • Peek: Top element-a mattum check panna without removing.
Python 
# Stack Example (LIFO)
stack = []
stack.append(10) # Push
stack.append(20)
print(stack.pop()) # Pop: Output 20 (Last In)

# Queue Example (FIFO)
from collections import deque
queue = deque([1, 2])
queue.append(3)
print(queue.popleft()) # Output 1 (First In)

Explanation: Kadaisi item-a mattum access panna Python-ku length theriyum so **O(1)** speed thaan.
