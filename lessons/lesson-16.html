2 Abstraction (Thevaiyanatha mattum kaattu)
S

Real-Life Example

Story
ATM Machine

Nee ATM center-ku pora. Screen-la "Withdraw" nu irukkum, atha amukkura. Kaasu varuthu. Machine kulla cash counting mechanism epdi irukku, server communication epdi pogudhunnu unakku theriya venam. Unakku Withdraw panna theriya thaan venum. Ithu thaan Abstraction — unnecessary complexity-a hide panni main functionality-a mattum kudukurathu.

Python 
from abc import ABC, abstractmethod

class Shape(ABC):
    @abstractmethod
    def area(self): pass

class Square(Shape):
    def __init__(self, s): self.s = s
    def area(self): return self.s * self.s

Quick Quiz – Security & Complexity

OOPs architecture master panniytiga-nu check pannalam!

1. '__' (Double underscore) vachi ezhutha variable-ku enna access level irukkum?

Public (Everyone)
Private (Only inside class)
Explanation: Triple-underscore illai, Double underscore thaan variables-a class ulla mattum use panna restrict pannum.

2. Data and Functions-a orey bundle-a wrap panratu enna?

Encapsulation
Abstraction
Explanation: Wrapping/Binding operation-kku **Encapsulation** nu peyar.

3. Abstraction use panna endha module-a import pannanum?

math
abc (Abstract Base Classes)
sys
Explanation: Python library-la irukkura abc module thaan abstraction-kku base.

4. Endha method-ku logic ezhuthama verum declaration mattum irukkumo athuku enna peyar?

Regular method
Abstract method
Explanation: Abstract method blueprint-la irukkum, Child class thaan actual logic ezhuthanum.

5. Real-life usage-la Encapsulation main goal enna?

Processing Speed
Data Security & Integrity
Explanation: Data-va yarum easy-a interfere pannaama **Security** kudukura thaan main reason.
