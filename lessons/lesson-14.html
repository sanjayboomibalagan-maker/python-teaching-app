OOPs: Classes & Objects

Object Oriented Programming — Python-oda "Building Blocks"-a blueprint vachi uruvaaku!

1 Class vs Object
S

Real-Life Example

Story
Car Blueprint vs Actual Car

Oru car company oru design (Blueprint) panni vachiruppanga. Athula car-ku 4 wheels, oru steering, and engine irukanum nu ezhudhi irukkum. Ithu thaan Class. Intha blueprint-a vachi factory-la senja actual car thaan Object. Blueprint onnu thaan, aana athai vachi different colors-la (Red car, Blue car) neraya actual cars-a senjukalam.

2 The __init__() and 'self'
C

Concept

Mechanism
  • __init__: Oru constructor function. Object create aagum pothu automatic-a run aagum.
  • self: Ithu current object-a reference pannum. Athavathu "Intha car-oda color", "Intha car-oda price" nu solra maari.
Python 
class Car:
    def __init__(self, name, color):
        self.name = name
        self.color = color

# Creating Objects
my_car = Car("Tata Nexon", "Blue")
friend_car = Car("Thar", "Black")

print(my_car.name)
