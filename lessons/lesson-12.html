2 The 'with' Statement (Best Practice)
C

Concept

Safety

Normal-a constant-a file-a open panna, vela mudicha apparam close() pannanum (Close panna marantha file corrupt aagalam). with statement use panna, Python-ae automatic-a safe-a close pannidum.

  • 'r': Read (Padikka).
  • 'w': Write (Ezhutha - pathasai erase pannidum).
  • 'a': Append (Kadaisiyila serkka).
Python 
# Ezhuthuthal (Writing)
with open("kavidhai.txt", "w") as f:
    f.write("Tamil is sweet!")

# Padithal (Reading)
with open("kavidhai.txt", "r") as f:
    content = f.read()
    print(content)

Quick Quiz – File Handling

Data-va safe-a store panna kathukitiga-nu check pannalam!

1. File-a handle panna first enna function call pannanum?

start()
open()
access()
Explanation: Python-la endha oru file operations-kum open() step thaan aarambam.

2. Old data-va delete pannaama, extra-va data serkka enna mode venum?

Write ('w')
Append ('a')
Explanation: 'w' mode already irukura content-a erase pannidum. 'a' mode thaan kadaisiyila extra-va add pannum.

3. File-a automatic-a close panna use aaguva keyword enna?

with
using
Explanation: with block vachi file open panna, vela mudinja udaney auto-close aagidum. Safety!

4. Default-a file-a enna mode-la open pannum (mention pannaalanna)?

Write ('w')
Read ('r')
Explanation: Nee mode mention pannaalana, Python automatic-a 'r' (Read) mode thaan eduthukkum.

5. 'with' keyword use pannaati, manual-a enna command kudukkanum?

f.stop()
f.close()
f.end()
Explanation: Memory release panna and file save aaga f.close() romba mukkiyam.
