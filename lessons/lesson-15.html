2 Polymorphism (Bahu-roopi)
S

Real-Life Example

Story
The "Push" Action

Nee oru elevator-la "Push" button-a amukuna lift varum. Nee oru doorbell-a "Push" pannuna sound varum. Nee oru mobile-a "Push" (Swipe) pannuna unlock aagum. Action ennavo "Push" thaan (Same function name), aana entha object thittu amukkiyo athuku etha maari result varum. Ithu thaan Polymorphism.

Python 
class Dog:
    def sound(self): print("Bark!")

class Cat:
    def sound(self): print("Meow!")

for animal in [Dog(), Cat()]:
    animal.sound()

🚀 Mastery Deep Dive: Advanced OOP Patterns

MRO (Method Resolution Order)

Multiple inheritance-la orey function name rendu parent kitta iruntha, endha parent first varuvaanga nu Class.mro() vachi paarkkalaam.

Operator Overloading

Rendu objects-a + panna enna nadakanum nu nee ezhuthalaam! Example: __add__ method.

def __add__(self, other): return self.x + other.x

isinstance()

Oru object endha class-a serndhathu nu check panna isinstance(obj, Class) use pannuvom.

Quick Quiz – Inheritance & Polymorphism

1. Inherit panna kudukkura original class-ku enna peyar?

Parent / Base class
Explanation: Sothu kudukkura original class-a Parent/Base class-nu solluvom.

2. Child class-la irundhu Parent class values-a access panna enna keyword venum?

base()
Explanation: super() use pannithaan parent class methods or __init__ a call pannuvom.

3. Oru class neraya classes kitta irundhu sotha vaanga mudiyuma (Multiple Inheritance)?

Illa, onnu mattum thaan
Explanation: Python supports Multiple Inheritance where a class can have more than one parent.

4. Polymorphism-kku literal meaning enna?

One form
Many forms
Explanation: Greek language-padi Poly = Many, Morph = Forms. Athavathu oru function different forms-la vela seiya mudiyum.

5. Parent method-a child class-la mathi ezhuthurathuku enna peyar?

Overloading
Method Overriding
Explanation: Already irukura parent logic-a overwrite pannurathuku Method Overriding nu peyar.
