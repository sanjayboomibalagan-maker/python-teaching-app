Software Testing (unittest)

Bug-free Code — Code-a deploy panna munnadiye ella logic-um correct-anu verify pannu.

1 Why Test?
S

Real-Life Example

Story
ISI Mark / Food Taster (Safety First)

Nee oru periya function-ku saapaadu pandra. Saapaadu ellarukum parimarura munnadi, oru plate-la konjama eduthu taste panni papom (Testing). Salt/Spice ellam correct-a irukka-nu double check pannina thaa function success aagum. Ithu thaan Unit Testing. Direct-a client-ku code-a kudukkaama, small tests vachi code and logic-a confirm pannanum.

💡 Quick Concept Check

Python-oda built-in testing framework peyar enna?

import
2 Assertions Logic
C

Concept

Verification
  • TestCase: Oru group of related tests-kulla irukara class.
  • assertEqual: (Expected Value == Result) nu check panna.
  • assertTrue: Oru condition True-a irukannu check panna.
Python 
import unittest

def add(a, b): return a + b

class MyTest(unittest.TestCase):
    def test_add(self):
        self.assertEqual(add(2, 3), 5)

if __name__ == '__main__':
    unittest.main()

Quick Quiz – Testing Power

Quality check logic purinjiducha-nu papom!

1. Output andha value-ku equal-a irukka-nu check panna endha method use pannanum?

verifyValue()
assertEqual()
Explanation: **assertEqual(a, b)** rendum same-anu check panni fail signal tharum if they differ.

2. Unittest-la classes create panna endha base class-a inherit pannanum?

unittest.TestCase
unittest.BaseTest
Explanation: **TestCase** thaan testing engine-oda heart.

3. Unittest function names ethula start aaganum?

test_
check_
verify_
Explanation: Unittest automatically **test_** nu start aagura functions-a mattum thaan run pannanum.

4. Command line-la irundhu tests-a trigger panna enna call pannanum?

unittest.run()
unittest.main()
Explanation: **unittest.main()** default-a command line trigger-a handle pannanum.

5. Oru test case-la vera logic check panna munnadi fail aayita, mitha tests run aaguma?

Yes, isolated-a run aagum
No, stop aayidum
Explanation: Oro test case-um **Isolated**. Onnu fail aanaalum mathathaiyum verify pannanum.
