Lesson 25

Virtual Environments (venv)

Project Isolation — Oru project library version innooru project-a break pannaama paathuko!

1 Why venv?
S

Real-Life Example

Story
Room Partition (Personal Space)

Nee oru chinna veetla irukinganu vechikuvom. Un thambi oru maari room-a decorate panna solran, nee vera maari solra. Rendhu perukkum "Conflict" aagum. Idhuku bathula oru "Partition" (Virtual Env) pottuta, un thambi avan pakkam pathuppan, nee un side pathupa. Python-laysm oru project-ku old version library venum, innooru project-ku new version venum na, venv thaan solution.

💡 Quick Concept Check

Python-la virtual environment create panna standard command enna?

python -m myenv
2 Pip & Requirements
C

Concept

Management
  • pip: Python packages-a install panna help pannum helper tool.
  • requirements.txt: Un project-ku endha library ellam venumnu ezhuthi vacha list.
  • Activate: Virtual env-kulla "Enter" aaga panna vendiya step.
Terminal 
# Create Env
python -m venv venv

# Activate (Windows)
.\venv\Scripts\activate

# Install library
pip install requests

# Save list
pip freeze > requirements.txt

Quick Quiz – Project Isolation

Environment logic purinjiducha-nu check pannalam!

1. Innooru per ezhuthuna Python package-a download panna endha tool venum?

python
pip
Explanation: **pip** (Package Installer for Python) thaan library download panna use aagum.

2. Virtual environment-oda main advantage enna?

Dependency Isolation
Run speed increase
Explanation: Oru project library mixed up aagame **Isolate** panni vakkum.

3. Project-ku thevaiyana libraries name-a endha file-la default-a store pannuvanga?

lib.txt
requirements.txt
Explanation: Standard-a **requirements.txt** file thaan dependencies-a share panna use pannuvom.

4. Environment-a create pannuna udanae adhu active aayiduma?

No, manually activate pannanum
Yes, automatic
Explanation: Create pannuna apram **Scripts/activate** command run pannanum.

5. Ippo install aagi irukura ella libraries list-ayum extract panna enna command?

pip list
pip freeze
Explanation: **pip freeze** exact versions-oda list-a tharum (requirements.txt file-ku idhu thaan best).
