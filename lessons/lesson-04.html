Lesson 04

Conditional Statements

if, elif, else — program-la logic and decisions edukka kaththukko

1 if, elif, else Basics
S

Real-Life Example

Story
Bus Ticket Fare Decision

Bus-la conductor kitta Ticket keka pora. IF nee chinna paiyan-na (child), half ticket. ELIF nee senior citizen-na, free/concession. ELSE (ithu ethuvume illana), full ticket. Condition-a poruthu operation maarum — ithu thaan Python-la conditional statements.

🚌
C

Concept

Understanding

Python-la syntax romba simple. Important-a colon (:) and Indentation (spaces) gavanikanum.

if mark >= 35:
Keyword if
Condition mark >= 35
Separator : (Colon)

💡 Pro Tip: The Indentation Rule

Python-la matha languages maari curly brackets {} illa. Athukku pathila block-a identify panna 4 spaces (Indentation) use pannuvom. Colon potta pothum, next line automatic-a thalli varum. Ithai mathuna error varum!

>

Worked Example

Code
Python 
age = int(input("Enter your age: "))

if age >= 18:
    print("Eligible to vote!")
elif age >= 13:
    print("Teenager")
else:
    print("Child")

🚀 Mastery Deep Dive: Truthiness & Ternary

"The Truth" in Python

Python-la 0, None, "" (empty string), [] (empty list) - ellaamey False thaan. Itha thavira ethu irunthaalum True.

if "": print("Hi")
Output: Nothing (Empty string is False)

Ternary Operator (One-liner If)

Professional codes-la if/else-a orey line-la ezhuthuvaanga.

status = "Adult" if age >= 18 else "Minor"
2 Logical Operators in Conditions
C

Multiple Conditions

Understanding

Orey line-la rendu moonu conditions check panna and, or, not use pannuvom.

Logic Condition Result
and A and B Rendum True-va irukanum
or A or B Ethavathu onnu True-va irundha pothum
not not A Reverse pannum (True becomes False)
Python 
mark = 85
attendance = 90

if mark > 80 and attendance > 75:
    print("Nee Distinction-la pass!")

if mark < 35 or attendance < 50:
    print("Nee fail, inimey ozhunga padi!")
3 Nested If
S

Innoru Condition Ulla Innoru Condition

Logic
ATM Transaction Example

IF nee card potta inside -> IF pin correct -> Cash withdraw pannalaam. Pin thappa iruntha Card block aagum. Inga condition-ulla innoru condition irukkuru thaan Nested If.

Python 
has_card = True
pin = 1234

if has_card:
    if pin == 1234:
        print("Login Success!")
    else:
        print("Wrong PIN!")
else:
    pass  # Innum code ezhuthala

🚀 Mastery Deep Dive: The pass Keyword

Python-la empty blocks allowance illa. Oru condition ezhuthitu apparam code ezhuthalaam nu ninaicha, empty-a vidakoodathu.

Why pass?

pass keyword use panna, Python antha block-a skip pannum. Ithu oru "Place Holder". NotImplementedError thavira ithu thaan professional approach.

!

Common Mistakes

Watch out
Mixing "=" and "=="
if age = 18: # Error: Assign panra, compare pannala if age == 18: # Correct: Check panra
Indentation Error (Tab/Space)
if True: print("Hello") # Spaces illa if True: print("Hello") # 4 spaces venum
Marakkura Colon (:)
if a > b # Colon enga? if a > b: # Correct
?
