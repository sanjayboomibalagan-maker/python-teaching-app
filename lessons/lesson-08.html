Lesson 08

Dictionaries

Key-Value Pairs — mapping and fast lookup concepts

1 Dictionary Basics
S

Real-Life Example

Story
Mobile Contact List (Phonebook)

Unga mobile contacts poidunga. Inga oru "Name" ezhuthirukkum, athukku ethura oru "Phone Number" irukkum. Name thaan Key, Number thaan Value. Index number use pannama, peyar-a vachi information edukkalam. Ithu thaan Dictionary.

📱
C

Concept

Understanding

Dictionary curly braces { } ulla key:value format-la ezhuthanam.

user = {"name": "Sanjay"}
Key "name"
Separator : (Colon)
Value "Sanjay"

💡 Pro Tip: Custom Keys

Dictionary-la keys unique-a irukanum. Orey key-a thirumba use panna, pazhaya value update aagidum.
D = {"a":1, "a":2} — ippa "a" oda value 2 thaan irukkum!

Python 
contact_list = {
    "Raja": 9876543210,
    "Sanjay": 8877665544
}

print(contact_list["Raja"])  # 9876543210 (Access by Key)
2 CRUD Operations (Create, Read, Update, Delete)
Just Keys.keys()
Just Values.values()
Both.items()
Safe Read.get()
Python 
user = {"name": "Arun", "city": "Chennai"}

# Create/Update
user["age"] = 25
user["city"] = "Madurai"

# Read with .get() (safe way)
print(user.get("job", "Not Found"))

# Delete
del user["age"]
user.pop("city")

Quick Quiz – Key-Value Check!

Dictionary concepts clear-a irukkanu check pannalam.

1. Dictionary create panna enna symbols use pannuvom?

[] Square brackets
{} Curly braces
() Parentheses

2. Dictionary-la identifier (name) part-a ennanu solluvom?

Value
Index
Key

3. Dictionay Keys eppadi irukkanum?

Duplicate-a irukalam
Unique-a irukanun (duplicates forbidden)

4. Oru key irukkanu safe-a check panna enna method best?

find()
get()
read()

5. Both Keys and Values rendaeyum ore loop-la edukka enna method venum?

.keys()
.values()
.items()
