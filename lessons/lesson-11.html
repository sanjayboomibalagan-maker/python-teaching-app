2 Different ways to Import
C

Concepts

Syntax
  • import module: Full library-a import pannum.
  • as: Periya name-a short-a mathikka (Alias).
  • from ... import: Specific tool-a mattum edukkalam.
Python 
# Method 1: Regular
import math
print(math.sqrt(16))

# Method 2: Alias (Nickname)
import datetime as dt
print(dt.datetime.now())

# Method 3: Specific tool
from random import randint
print(randint(1, 100))

Quick Quiz – Modules & Imports

1. Oru module-a load panna enna keyword venum?

Explanation: Python-la libraries use panna standard-aana keyword import.

2. Module name-ku 'nickname' (alias) kudukka enna ezhuthanam?

name

Explanation: import math as m — ithula as thaan math-ku short name fix pannum.

3. Module-la irundhu specific-a oru tool-a mattum edukka enna use pannalam?

Explanation: from random import randint maari from keyword use panni specific parts edukkalam.

4. Module ulla irukura function-a call panna enna symbol use aagum (e.g., math_sqrt)?

Explanation: math.sqrt() maari dot (.) operation thaan functions call panna use aagum.

Explanation: Ellaa Python files-um .py extension-la thaan irukka pannanum.
