1 print() Function
S

Real-Life Example

Story
Tea kadai-la order solla

Nee oru tea kadai-ku poi, counter-la nikkura. Nee enna pannuva? "Oru tea kudunga" nu solluva, illaya? Nee sollama iruntha, kadaikkaaran unakku enna venum nu theriyaathu.

Atha ponnu kadaikkaaran unakku tea kudukkanum na, nee solli thaan aaganum. Antha "solla" — athaan programming-la print().

Computer-kku nee enna venum nu sollanumna, print() use pannanum. Nee sollaama iruntha, computer screen-la onnum varaathu. Tea kadai-la nee vaai thirakaama ninna maari — useless.

C

Concept Explanation

Understanding

print() is a built-in function in Python. Ithu enna pannum? Nee kudukkura text-a screen-la display pannum. Adhaan ithu panna oru velai.

Nee enna text kudukkanum-o, atha quotation marks ("..." or '...') ulle podu. Aprom atha print()-kku kudu. Python atha screen-la print pannidum.

Ithu verum display function. Idhu ethayum calculate pannaathu, ethayum store pannaathu. Verum kaatthidum, adhu thaan velai.

$

Python Syntax

How to write it

General format:

Python 
print("Un message idhula podu")
  • print — ithu function name. Lowercase la ezhuthanum, Print or PRINT vena
  • () — round brackets. Function-a call pannura sign
  • "..." or '...' — un text-a ithula wrap pannu. Rendu-la ethu use pannaalum orey result thaan
>

Worked Example

Code + Output
Python 
print("Vanakkam, Python!")
print("Naan Python kaththukittu irukken.")
print("2 + 3 =", 2 + 3)
Output
Vanakkam, Python!
Naan Python kaththukittu irukken.
2 + 3 = 5

Line 1: Simple text print aagum. Line 2: Intha text print aagum. Line 3: Text-um number calculation-um combine pannirukkum. 2 + 3 nu Python calculate panni 5 nu result kudukkum.

Comma (,) use pannina, print() oru space vitta print pannum. "2 + 3 =" aprom space aprom 5.

2 Variables
S

Real-Life Example

Story
Dabba-la label potta store panra maari

Un amma kitchen-la different dabbas vechirukkanga. Oru dabba-la "Arisi" nu label irukkum, innum oru la "Paruppu" nu irukkum, vera onnula "Sakkarai" nu irukkum.

Dabba-thaan variable. Label-thaan variable name. Ulle irukka poruluthaan value.

Amma "paruppu kudu" nu sonna, nee "Paruppu" label irukka dabba-va edukkuva. Athe maari, Python-la nee oru name kuduththu, oru value store pannu. Aprom antha name solli antha value-a use pannu.

Dabba-la irukka porula maathalam — arisin-a eduthu, paruppu-la poodalam. Athe maari, variable-la irukka value-yum maathalam.

C

Concept Explanation

Understanding

Variable means oru named storage. Nee oru value-a oru name-la store pannuva. Python-la variable create panna, = sign use pannu. Ithukku assignment operator nu per.

Left side-la name podu, right side-la value podu. Python automatic-a type decide pannidum — nee solla vendaam.

Variable name rules:

  • Letter or underscore (_) la start aaganum. Number la start aaga koodaathu
  • Letters, numbers, underscore maththum use pannu. Spaces, special characters koodaathu
  • Python reserves panna keywords (if, for, print etc.) variable name-a use panna koodaathu
  • Case-sensitive: Name and name are different variables
$

Python Syntax

How to write it
Python 
variable_name = value

# Examples:
peyar = "Sanjay"
vayasu = 21
vilai = 49.50
is_student = True
>

Worked Example

Code + Output
Python 
# Details store pannurom
kadai_peyar = "Murugan Tea Stall"
tea_vilai = 15
bun_vilai = 10
total = tea_vilai + bun_vilai

print("Kadai:", kadai_peyar)
print("Tea vilai:", tea_vilai)
print("Bun vilai:", bun_vilai)
print("Total:", total)
Output
Kadai: Murugan Tea Stall
Tea vilai: 15
Bun vilai: 10
Total: 25

kadai_peyar la text store pannirukkum. tea_vilai and bun_vilai numbers store pannirukku. total la rendu vilai-yum koottirukkom. Variables use panni, reusable-a code ezhuthirukkom. Tomorrow tea vilai 20 aana, tea_vilai = 20 nu maathna podhum, baaki code automatic-a correct-a run aagum.

3 input() Function
S

Real-Life Example

Story
Bank-la form fill panra maari

Nee bank-ku pona, counter-la irukka person unna kelppaan: "Unga peyar enna?". Nee solluvaa "Sanjay" nu. Aprom kekkuvaan: "Phone number?" — nee solluvaa "9876543210" nu.

Avan un answer-a form-la ezhuthuvan. Avan kelvi kekkuraan, nee badhil solluva, avan atha store pannuvan.

input() function athaan ithu maari. Python unna oru question kekum, nee answer type pannuva, Python antha answer-a oru variable-la store pannidum.

C

Concept Explanation

Understanding

input() function user-kitta data vaangum. Program run aagum bodhu, screen-la oru message kaatum (nee kudukkura message), aprom user type pannum varai wait pannum.

Mukkiyam: input() எப்போவும் string return pannum. Nee number type pannaalum, adhuvum string thaan varum. Number-a use panna, nee convert pannanum (int() or float() use pannu).

$

Python Syntax

How to write it
Python 
# Text input vaanga:
variable = input("Un kelvi idhula podu: ")

# Number input vaanga:
variable = int(input("Number kelunga: "))

# Decimal number input vaanga:
variable = float(input("Decimal kelunga: "))
>

Worked Example

Code + Output
Python 
# Auto-la meter reading kekka maari
peyar = input("Unga peyar enna: ")
ooru = input("Enga poganum: ")
km = float(input("Evvalavu km: "))
rate = 15
total = km * rate

print("Driver:", peyar)
print("Destination:", ooru)
print("Total fare: Rs.", total)
Output (user "Sanjay", "T.Nagar", "8.5" nu type panna) 
Unga peyar enna: Sanjay
Enga poganum: T.Nagar
Evvalavu km: 8.5
Driver: Sanjay
Destination: T.Nagar
Total fare: Rs. 127.5

peyar and ooru text-a direct-a vaangirukkum. km la decimal number vaanganum, adhaan float() use pannirukkum. rate fixed value. total la km * rate panni calculate pannirukkum. Aprom print() use panni result kaattirukkum.

4 Data Types
S

Real-Life Example

Story
Ration card-la details maari

Government ration card-la enna irukkum? Un peyar (Sanjay — text), un vayasu (21 — whole number), un height (5.8 — decimal number), ration card holder-ah illayah (Yes or No).

Different type data, different-a store pannanum. Peyar-a number-la store panna mudiyuma? Mudiyaathu. Vayasu-la "Sanjay" nu store pannalaam-a? Logic-e illai.

Atha maari Python-layum, different types of data irukkum. Every value-kkum oru type irukkum. Python automatically type decide pannum, aana nee therinjukkanum enna type irukku nu.

C

Concept Explanation

Understanding

Python-la 4 main basic data types irukku:

  • str (String) — Text data. Quotes ulle ezhuthurathellam string. Example: "Vanakkam", 'Chennai'
  • int (Integer) — Whole numbers. Decimal point illai. Example: 10, -5, 0
  • float (Floating-point) — Decimal numbers. Example: 3.14, -0.5, 100.0
  • bool (Boolean) — True or False maththum. Yes/No type decisions-ku use aagum. Example: True, False

type() function use panni, oru value enna type-nu check pannalam.

$

Python Syntax

How to write it
Python 
# String
peyar = "Sanjay"

# Integer
vayasu = 21

# Float
height = 5.8

# Boolean
is_student = True

# Type check pannalam
print(type(peyar))      # <class 'str'>
print(type(vayasu))     # <class 'int'>
print(type(height))     # <class 'float'>
print(type(is_student)) # <class 'bool'>
>

Worked Example

Code + Output
Python 
# Ration card details
peyar = "Murugan"
vayasu = 45
weight = 72.5
card_holder = True

print("Name:", peyar, "- Type:", type(peyar))
print("Age:", vayasu, "- Type:", type(vayasu))
print("Weight:", weight, "- Type:", type(weight))
print("Card holder:", card_holder, "- Type:", type(card_holder))

# Type conversion
age_text = "25"
age_number = int(age_text)
print("Converted:", age_number + 5)
Output
Name: Murugan - Type: <class 'str'>
Age: 45 - Type: <class 'int'>
Weight: 72.5 - Type: <class 'float'>
Card holder: True - Type: <class 'bool'>
Converted: 30

type() use panni each value enna type-nu confirm pannalam. age_text la "25" string-a irukku. Atha int() use panni number-a convert panni, aprom 5 koottirukkom. Convert pannaaaa, "25" + 5 error varum.

!

Common Mistakes Beginners Make

Watch out
Mistake 1: Quotes marakka (Forgetting quotes in print)

String-a print pannumpodhu, quotes poodama iruntha, Python atha variable name nu ninaikkum.

print(Vanakkam) # NameError: name 'Vanakkam' is not defined print("Vanakkam") # Correct: quotes poddaachi
Mistake 2: Print nu capital letter la ezhuthurathu

Python case-sensitive. print maththum correct. Print, PRINT ellam error varum.

Print("Hello") # NameError: name 'Print' is not defined print("Hello") # Correct: lowercase print
Mistake 3: input() return-a direct-a math-la use panrathu

input() always string return pannum. Atha number-a convert pannama math operation panna error varum.

age = input("Vayasu: ") print(age + 5) # TypeError: can only concatenate str to str age = int(input("Vayasu: ")) print(age + 5) # Correct: int() la convert pannachi
Mistake 4: Variable-la space use panrathu

Variable name-la space use panna koodaathu. Underscore (_) use pannu.

my name = "Sanjay" # SyntaxError my_name = "Sanjay" # Correct: underscore use pannachi
Mistake 5: = and == mix-up

= means assign (value store pannu). == means compare (same-a nu check pannu). Beginners rendaiyum confuse pannuvaa.

x = 10 # Assign: x-la 10 store aagum x == 10 # Compare: x 10-a nu check pannum (True/False varum) Assign panna = use pannu. Compare panna == use pannu. Rendu purpose-um vera.

Q3: User-kitta peyar and birth year vaangi, avanga age calculate panni print pannu.

Medium

Q4: User-kitta Celsius temperature vaangi, atha Fahrenheit-la convert pannu.

Formula: F = (C * 9/5) + 32

Expected Output
Celsius temperature kudu: 37
37.0 Celsius = 98.6 Fahrenheit
Show Answer
Python 
celsius = float(input("Celsius temperature kudu: "))
fahrenheit = (celsius * 9/5) + 32

print(celsius, "Celsius =", fahrenheit, "Fahrenheit")
Real-Life Problem

Q5: Auto fare calculator — User-kitta starting place, ending place, km distance vaangu. Base fare = Rs.25, per km rate = Rs.12. Total fare calculate panni bill format-la print pannu.

Expected Output
===== AUTO BILL =====
From: Tambaram
To: Guindy
Distance: 18.0 km
Base fare: Rs. 25
Per km charge: Rs. 216.0
---------------------
Total fare: Rs. 241.0
=====================
Show Answer
Python 
start = input("Starting place: ")
end = input("Ending place: ")
km = float(input("Distance in km: "))

base_fare = 25
per_km_rate = 12
km_charge = km * per_km_rate
total = base_fare + km_charge

print("===== AUTO BILL =====")
print("From:", start)
print("To:", end)
print("Distance:", km, "km")
print("Base fare: Rs.", base_fare)
print("Per km charge: Rs.", km_charge)
print("---------------------")
print("Total fare: Rs.", total)
print("=====================")
